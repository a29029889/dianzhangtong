## 📊 项目概览

店账通是一款专为中小店铺设计的轻量级记账与库存管理 SaaS 系统，采用现代化的技术栈，提供跨平台支持（H5、微信小程序、App）。

### 核心价值
- 简单易用的记账功能
- 实时数据统计与可视化
- 多店铺支持
- 跨平台访问

## 📱 页面结构

   - 分类占比

4. **shop** - 店铺管理

## 🚀 部署说明

详细的部署指南请参考 [DEPLOY.md](./DEPLOY.md)

### 快速部署
1. **后端**: `npm install && npm run build && npm run start:prod`
2. **前端 H5**: `npm install && npm run build:h5`
3. **微信小程序**: `npm install && npm run build:mp-weixin`

### 环境要求
- Node.js 18+
- PostgreSQL 14+
- Redis 6+
